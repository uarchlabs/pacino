// overview
"I'm out of order? You're out of order!"

Pacino is an open source, AI co-designed 8-issue out-of-order RISC-V processor targeting the RVA23 profile. RTL is written in SystemVerilog and simulated with Verilator. The project documents not just the design itself, but the full AI-assisted co-design methodology — prompts, experiment loops, and architectural rationale — end to end.

// ai co-design methodology

Pacino is designed using a two-tool AI workflow. claude.ai provides architectural guidance and generates structured experiment prompts. Claude Code implements RTL and runs Verilator in the terminal. Neither tool communicates directly — the engineer bridges them, maintaining full control over every decision.

Every design decision is captured in an experiment file with a hypothesis, session prompt, results, and graduation status. Confirmed decisions are promoted to CLAUDE.md, the project baseline read by Claude Code at the start of every session. The full methodology — prompts, handoffs, and rationale — is committed alongside the RTL.
