  longest-history table with hit_p1[s] asserted.
  Fall back to T0 if no tagged table hits.
- Alternate provider: longest-history tagged table
  with hit_p1[s] asserted that is shorter than the
  primary provider. T0 is alternate when no tagged

  When triggered and uaon[s] MSB is set, use

  prediction matches alternate direction. Decrement
  unconditionally on tage_pred_rdy_p2[s] when not
  triggered. 4b saturating in both directions.

  Direction is CTR[1] (MSB).

    Direction is always taken from t_taken_p1[0][s], not the
