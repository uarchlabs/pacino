## Background

This is an 8-issue OoO RVA23 processor branch predictor cluster.
The bp_pkg.sv package is the single source of truth for all BP
cluster types, parameters, and structs. All subsequent BP modules
will import this package. It must be complete and internally
consistent before any implementation RTL is written.

The design follows an interface-first methodology. This session
produces bp_pkg.sv only. No implementation modules.

The predictor hierarchy is:
  uBTB, Loop predictor, FTB, TAGE, SC, ITTAGE, RAS
  (uBTB and Loop fire at s1; FTB/TAGE/RAS at s2; SC/ITTAGE at s3)

The FTQ entry is split into two structs:
  bp_ftq_entry_t  -- fast path, read every cycle
  bp_ftq_meta_t   -- slow path, read on update only

## Binding Previous Decisions

bp_cluster.md. Do not deviate.

- GHR_WIDTH         = 256

- FETCH_BLOCK_BYTES = 64

All other parameters (TAGE table sizes and entry layout,
SC table sizes, ITTAGE entry counts, loop predictor
parameters) are fully specified in bp_cluster.md. Read
that file first -- do not re-derive or guess these values.
