    ITAGE_RESP_BUF_DEPTH. Entry type
    ittage_pred_meta_t per slot. Bypass when
    empty and consumer_ready.

ITTAGE has no SC chaining. There is no downstream
chained predictor consuming the response buffer.
consumer_ready is tied to 1'b1 internally. It is
not a port.

ITTAGE has no redirect signals. Redirect
comparison is a bp_cluster responsibility.
