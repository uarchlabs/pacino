predictor. It drives all tage_table inputs, collects
tage_table outputs, applies provider selection and UAON
mux logic, and produces tage_pred_meta_t outputs.

logic is added in BP-008a-2. Update logic is added in

signals are always present unconditionally. Slot
dimension uses vector index [0:NUM_PRED_SLOTS-1]
throughout. There are no scalar per-slot signal pairs.

TAGE has five tables. T0 is the base table. T1-T4 are
tagged tables. tage_cntrl connects to all five.

  vector index [0:NUM_PRED_SLOTS-1]. No slot suffix in

### Gap: t_alc_tbl_sel_u0 in spec but not in tage_table.sv

tage_table_interfaces.md specifies alc_tbl_sel_u0 as a
separate selector port. tage_table.sv does not have this
port -- it gates alc_wr using prm_tbl_sel_u0 comparison.
tage_cntrl.sv declares t_alc_tbl_sel_u0 per the spec.
At TAGE top level wiring this must be reconciled: either
add alc_tbl_sel_u0 to tage_table.sv or remove it from
tage_cntrl.sv.

### Gap: t_alc_index_u0 in spec but not in tage_table.sv

tage_table_interfaces.md specifies alc_index_u0 as a
separate allocation address port. tage_table.sv uses
upd_index_u0 for the allocation RAM address. tage_cntrl.sv
declares t_alc_index_u0 per the spec. Must be reconciled
before TAGE top level integration.
