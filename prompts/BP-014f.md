### Address derivation notes

Formula (folded_hist=0 in all tests):
  idx_hash = (pc >> INST_OFFSET)[10:0] where INST_OFFSET=2
  bank = idx_hash[10]
  row  = idx_hash[9:0]
  tag_hash = (pc >> THIS_INDEX_BITS)[7:0] where
             THIS_INDEX_BITS=11

Tag formula verified from tage_table.sv lines 178-182:
  tag_hash = THIS_TAG_BITS'(
    (pc >> THIS_INDEX_BITS) ^ fh1_ext ^ (fh2_ext<<1))
  With fh=0: tag = (pc >> 11)[7:0]

T0 uses the same index formula as T1-T4 (pc[12:2]),
so bank and row are identical for T0 and T1 at a given PC.
