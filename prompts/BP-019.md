## Background

### Debt #27 -- bw_ram BANKS=2 magic number

In tage_bim.sv and tage_table.sv:
  localparam int RAM_ENTRIES = RAM_DEPTH / 2;
  bw_ram #(.ENTRIES(RAM_ENTRIES), .BANKS(2)) u_ram_s0
  bw_ram #(.ENTRIES(RAM_ENTRIES), .BANKS(2)) u_ram_s1

The literal 2 appears three times per file (RAM_DEPTH/2
and two .BANKS(2) instantiations). It should be a named
localparam so the relationship between RAM_ENTRIES and
BANKS is explicit and a future bank count change only
requires one edit.

### Debt #24 -- TAGE_TBL_* scalar parameter cleanup

Four scalar parameters in bp_defines_pkg.sv are marked
FIXME and have named consumers that should use canonical
alternatives:

  TAGE_T0_CTR_BITS (= 2):
    Consumer: tage_cntrl.sv
    Uses: u_prm_ctr[s][TAGE_T0_CTR_BITS-1:0]
          u_prm_ctr[s][TAGE_T0_CTR_BITS:1]
    Replace with: TAGE_TBL_CTR[0]
    Rationale: TAGE_TBL_CTR[0] is the authoritative
      per-table CTR width for T0. These are independent
      parameters that happen to share a value.

  TAGE_TAG_BITS (= 8):
    Consumer: tage_cntrl.sv
    Uses: localparam int ALLOC_DATA_WIDTH =
            CNTRL_BITS_WIDTH + TAGE_TAG_BITS
    Replace with: MAX_TAG_WIDTH
    Rationale: MAX_TAG_WIDTH is the canonical max tag
      field width used on all ports and buses. These
      are independent parameters that happen to share
      a value.

  TAGE_CTR_BITS (= 3):
    Consumer: bp_structs_pkg.sv
    Uses: tage_pred_meta_t fields tage_prm_ctr and
          tage_alt_ctr: logic [TAGE_CTR_BITS-1:0]
          Also in deprecated bp_tage_meta_t.
    Replace with: MAX_CTR_WIDTH
    Rationale: MAX_CTR_WIDTH is the canonical max CTR
      width used on all tage_table ports and buses.
      These are independent parameters that happen to
      share a value.

  TAGE_USEFUL_BITS (= 2):
    Consumer: bp_structs_pkg.sv
    Uses: tage_pred_meta_t fields tage_prm_useful and
          tage_alt_useful: logic [TAGE_USEFUL_BITS-1:0]
          Also in deprecated bp_tage_meta_t.
    Replace with: MAX_USE_WIDTH
    Rationale: MAX_USE_WIDTH is the canonical max USE
      width used on all tage_table ports and buses.
      These are independent parameters that happen to
      share a value.

After replacing all consumers, the four scalar parameters
have no remaining references and can be removed from
bp_defines_pkg.sv.
